# WBS Editor - 工作分解結構編輯器

一個基於 React + Firebase 的 WBS (Work Breakdown Structure) 編輯器，支持樹狀結構的項目分解、協同編輯。

## 功能特色

- 📋 **樹狀結構** - 無限層級的項目分解
- ✏️ **內聯編輯** - 直接在節點上編輯，無需彈窗
- 👥 **負責人管理** - 為每個任務指派負責人
- ⏱️ **工時估算** - 記錄預估工時
- 📅 **時間規劃** - 設定開始/結束日期
- 🔄 **即時同步** - Firebase 實時數據庫
- 🔐 **房間密碼** - 簡單的訪問控制

## 使用方式

### 新增項目
- 點擊「+ 新增階段」創建根節點
- 點擊節點的「+ 子項目」按鈕添加子節點

### 編輯項目
- 點擊「✏️ 編輯」按鈕進入編輯模式
- 可修改：名稱、負責人、工時、開始日期、結束日期
- 點擊「✓ 儲存」保存，「✕ 取消」放棄修改

### 展開/收合
- 點擊節點前的 ▼ 箭頭可展開或收合子節點

### 刪除項目
- 點擊「🗑️」按鈕刪除項目
- 刪除父節點會同時刪除所有子節點

## 技術架構

- **前端框架**: React 18
- **建置工具**: Vite
- **資料庫**: Firebase Realtime Database
- **部署**: Zeabur / Vercel

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發服務器
npm run dev

# 建置生產版本
npm run build
```

## 環境變數
